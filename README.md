# 簡単なPython仮想環境のセットアップ

このリポジトリには、Pythonの仮想環境を簡単にセットアップするための`setup.cmd`および`setup‐v2.cmd`スクリプトが含まれています。以下は、このスクリプトの使用方法について説明します。
なお`setup‐v2.cmd`は改良中のため基本的には`setup.cmd`を使用することを推奨しています。
将来的に`setup‐v2.cmd`は`setup.cmd`に統合する予定です。
## スクリプトの内容

`setup.cmd`は以下の手順を自動化します：

1. 仮想環境が存在しない場合は作成します。
2. 仮想環境のアクティベート方法を表示します。
3. Pythonスクリプトの実行方法を説明するメッセージを表示します。
4. 実行後のオプションを提示します。

   - 終了する
   - 現在いるディレクトリでVSCodeを開く

## 使用方法
1. **ダウンロード**
   右上の```code```を押して```DownloadZip```を選択してください。  
   その後Zipファイルを解凍し、実行したいPythonフォルダがある場所へ`setup.cmd`を移動させてください。
2. **スクリプトの実行**
   - `setup.cmd`をダブルクリックして実行します。
3. **仮想環境のセットアップ**
   - スクリプトは自動的に`venv`のフォルダを作成し、仮想環境をセットアップします（存在しない場合）。
4. **仮想環境の有効化**
   - スクリプト内で仮想環境を有効化します。
5. **Pythonスクリプトの実行に関する説明**
   - 次の手順が表示されます：

      ```
      1. 仮想環境を有効化する: venv\Scripts\activate
      2. 必要なモジュールをインストールする: pip install <module_name>
      3. Pythonスクリプトを実行する: python <script_name>.py
      ```
6. **実行後のオプション**
   - `1`: スクリプトを終了します。
   - `2`: 仮想環境を有効化した状態で、現在のディレクトリをVSCodeで開きます。
     ```
     code .
     ```

## 必要要件

- Python3 がインストールされていること
- [Visual Studio Code (VSCode)](https://code.visualstudio.com/)がインストールされていること（オプション）

## 注意事項

- スクリプト実行前に、仮想環境を作成したいディレクトリに必ず移動してください。
- `setup.cmd`をデフォルトで使うため、Windows環境が必要です。
